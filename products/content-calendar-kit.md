## Part 1: Content Strategy Foundation

### Define Your Content Pillars

Choose 4-5 themes to rotate through. This keeps your content consistent and your audience engaged.

**Example Pillars for a Business/Creator Account:**

| Pillar | % of Content | Purpose | Example |
|--------|-------------|---------|---------|
| Educational | 40% | Build authority | Tips, tutorials, how-to's |
| Behind-the-Scenes | 20% | Build connection | Process, daily life, workspace |
| Promotional | 20% | Drive revenue | Product features, offers, launches |
| Community | 15% | Build engagement | Polls, Q&A, reposts |
| Inspirational | 5% | Build brand love | Quotes, milestones, stories |

### Brand Voice Guidelines

| Element | Your Brand |
|---------|-----------|
| Tone | [Professional / Casual / Bold / Friendly] |
| Language Level | [Simple / Technical / Conversational] |
| Humor | [None / Light / Frequent] |
| Emoji Use | [None / Minimal / Moderate / Heavy] |
| Perspective | [First person / Brand voice / Third person] |

### Platform-Specific Best Practices

**Instagram:**
- Best times: Tue-Thu 10am-1pm, Fri 11am
- Reels: 15-30 seconds perform best
- Carousel: 7-10 slides ideal
- Stories: 3-7 per day, use polls/questions

**Twitter/X:**
- Best times: Mon-Fri 8-10am, 12-1pm
- Threads: 5-15 tweets, bold hook first
- Engage: Reply to others 15 min before/after posting
- Frequency: 3-5 tweets/day

**LinkedIn:**
- Best times: Tue-Thu 7-8am, 12pm, 5-6pm
- Document posts (carousels) get 3x engagement
- Personal stories outperform corporate content
- Frequency: 1x/day weekdays

**TikTok:**
- Best times: Tue 2-3pm, Thu 12pm, Fri 3pm
- Hook in first 2 seconds
- Trending sounds boost reach 50%+
- Frequency: 1-3x/day

## Part 4: Content Repurposing Framework

### The 1 → 8 Method

**Start with 1 long-form piece** (blog post, podcast, or video) and transform it into:

| Output | Platform | How |
|--------|----------|-----|
| Twitter/X Thread | Twitter | Extract 10 key points, each = 1 tweet |
| LinkedIn Post | LinkedIn | Write a personal angle on the topic (200 words) |
| Instagram Carousel | Instagram | Turn key points into 8-10 visual slides |
| Reel/TikTok | Instagram/TikTok | Film a 60-second summary with a hook |
| Email Newsletter | Email | Summarize with personal commentary and CTA |
| Quote Graphics | Instagram/Twitter | Pull 3-5 quotable lines, put on branded backgrounds |
| Reddit Post | Reddit | Reframe as a question/discussion for relevant subreddit |
| Pinterest Pin | Pinterest | Create an infographic or checklist visual |

### Platform Adaptation Guide

| Element | Instagram | Twitter/X | LinkedIn | TikTok |
|---------|-----------|-----------|----------|--------|
| Tone | Visual + casual | Witty + concise | Professional + personal | Authentic + trendy |
| Length | Caption: 100-200 words | 280 chars / thread | 200-300 words | 15-60 seconds |
| CTA | "Save this" / "DM me" | "Retweet" / "Follow for more" | "What do you think?" | "Follow for Part 2" |
| Best Format | Carousel / Reel | Thread / Quote tweet | Text + document | Talking head / trend |

## Part 6: AI Content Generation Prompts

### Prompt 1: Weekly Content Batch Creator
```
Act as a social media strategist. Generate 5 social media posts for [brand/niche] for this week. For each post include: platform (Instagram/Twitter/LinkedIn), content type, hook/headline, full caption or thread, 10 relevant hashtags, and best posting time. Content pillars to rotate: [list your 4 pillars]. Tone: [your brand tone].
```

### Prompt 2: Carousel Content Writer
```
Create an Instagram carousel post about [topic]. Write: Slide 1 (hook — make them stop scrolling), Slides 2-8 (one key point per slide, large text format), Slide 9 (summary), Slide 10 (CTA). Each slide should have a headline (5-8 words) and optional body text (under 20 words). Caption: 100-200 words with 5 hashtags.
```

### Prompt 3: Twitter Thread Generator
```
Write a Twitter/X thread about [topic] with 12 tweets. Tweet 1: bold hook that creates curiosity. Tweets 2-10: one insight per tweet, mix data points and personal observations. Tweet 11: actionable takeaway. Tweet 12: CTA (follow + check out [link]). Each tweet under 280 characters. No hashtags except in the final tweet (3 max).
```

### Prompt 4: LinkedIn Post Writer
```
Write a LinkedIn post about [topic/experience]. Start with a bold first line that hooks readers. Use short paragraphs (1-2 sentences each). Include a personal story or lesson. End with a question to drive comments. Total: 200-300 words. Tone: professional but personal. No hashtags in body, add 3-5 at the end.
```

### Prompt 5: Content Idea Generator
```
Generate 20 content ideas for [niche/brand] across these pillars: Educational (8), Behind-the-Scenes (5), Promotional (4), Community (3). For each idea, provide: headline, content type (reel/carousel/thread/post), platform, and a one-sentence description. Focus on ideas that drive engagement and are relevant to [target audience].
```

### Prompt 6-10: Reel Scripts, Story Ideas, Caption Writer, Hashtag Research, Analytics Insights
